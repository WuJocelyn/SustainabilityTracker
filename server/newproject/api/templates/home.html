Welcome to the SustainabilityTracker Backend API

This service provides endpoints for managing actions and retrieving related analytics. Below is a list of available API routes.

Available Endpoints

  • GET /api/actions/
    Retrieve a list of all actions.

  • POST /api/actions/create/
    Create a new action.

  • GET /api/actions/<id>/
    Retrieve details for a specific action by ID.

  • GET /api/actions/points_timeseries/
    Retrieve a points time-series for analytics.
